Этот проект состоит из двух частей:

Веб-приложение (app.py): Сайт для обработки и управления базой данных.

GUI-приложение (gui.py): Десктопное приложение для работы с базой данных, включая аналитику.

1)Установка

Убедитесь, что у вас установлен Python 3.8 или выше.
Клонируйте репозиторий.

2)Основные библиотеки

Flask: Веб-фреймворк для веб-приложения (app.py).
pip install Flask

SQLAlchemy: ORM для работы с базой данных.
pip install SQLAlchemy

PyQt5: Библиотека для создания GUI-приложения (gui.py).
pip install PyQt5

3)Скачивание базы данных

База данных не включена в репозиторий(github не дает загрузить файл больше 25mb). Её нужно скачать отдельно:
Перейдите по ссылке для скачивания базы данных: https://www.kaggle.com/datasets/kaggle/us-baby-names.
Скачайте файл базы данных (database.sqlite).
Поместите файл базы данных в папку проекта.

4)Запуск приложений

Веб-приложение (app.py)
Перейдите в папку проекта:
cd ваш-репозиторий
Запустите Flask-приложение:
python app.py
Откройте браузер и перейдите по адресу:
http://127.0.0.1:5000/

GUI-приложение (gui.py)
Перейдите в папку проекта:
cd ваш-репозиторий
Запустите GUI-приложение:
python gui.py

5)Функциональность

Веб-приложение (app.py)
Добавление записей: Добавление новых записей в базу данных.
Удаление записей: Удаление записей по ID.
Поиск записей: Поиск записей по имени.
Отображение данных: Просмотр первых 50 записей.

GUI-приложение (gui.py)
Добавление записей: Добавление новых записей в базу данных.
Удаление записей: Удаление записей по ID.
Поиск записей: Поиск записей по имени.
Отображение данных: Просмотр первых 50 записей.
Аналитика: В gui.py добавлена аналитика, которая отображает распределение по полу (мужчины/женщины) в таблицах NationalNames и StateNames. Это позволяет пользователю видеть процентное соотношение мужских и женских имен в базе данных.

6)Структура проекта

app.py: Основной файл Flask-приложения.
gui.py: Основной файл GUI-приложения.
orm.py: Модели SQLAlchemy для работы с базой данных.
crud.py: Функции для CRUD-операций.
templates/: HTML-шаблоны для веб-приложения.
	index.html: Главная страница.
database.sqlite: База данных SQLite (не включена в репозиторий, нужно скачать).
